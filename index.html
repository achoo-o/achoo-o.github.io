About this website

This website was created to keep my coding knowledge alive and use it to fuel my interest in learning the Māori language. This website will be used to collate all the free resources I find and use whilst beginning my journey.

Sidenote: The links in the header are designed to be used in (somewhat) chronological order.

Throughout my learning I will also use Anki (also available for mobile devices). It's basically a place to create digital flashcards to memorise vocabulary. I will be working on making a page dedicated to any word lists I create which can be put directly into Anki. I will also be working on creating a page filled with alternative resources to learning vocabulary.

Focusing on vocabulary is not my main goal, however. Whilst important, I believe the key to learning a language is understanding structure; how to manipulate things like verbs and adjectives to create more complex sentences. Of course, listening to people speak the language and practising pronunciation is also important, so I will try to work that in as well.
